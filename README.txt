BEE ADVENTURE
This is a simple Alexa Skill using AWS Lambda and the Alexa SDK to demonstrate the echo's usability as a live action role-playing game. 

SETUP
This project is not published under Amazon's Alexa Skills and must be manually imported into Lambda and configured at developer.amazon.com for use. The main code is in JS (using json) and is handleled in beeAdenvture.js

For more information on how to set up an Alexa Skill, please refer to any of the README's in Amazon's sample Alexa Skills.
